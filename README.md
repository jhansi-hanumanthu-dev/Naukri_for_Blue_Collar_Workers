# JobMitra - React Job Portal

## Overview
JobMitra is a responsive job portal built with **React** and **TypeScript**.  
It allows users to:  
- Select a language (English, Hindi, Telugu)  
- Browse job categories  
- View jobs in each category  
- See detailed job information and apply  

---

## Tech Stack
- React  
- TypeScript  
- Zustand (state management)  
- React Router DOM  
- CSS (responsive design)  

---

## Installation
1. Clone the repo:
git clone https://github.com/jhansi-hanumanthu-dev/Naukri_for_Blue_Collar_Workers.git
cd JobMitra
2.Install dependencies:
npm install

3.Run the app:
npm start
Open http://localhost:3000 in your browser.

## Thought Process & Achievements
How I thought about the problem:

Users searching for jobs often see many different titles for the same kind of work, making it hard to find relevant results.

JobMitra categorizes jobs and makes browsing easy across multiple job titles and languages.

Technology and design choices:

Used React + TypeScript for a robust, type-safe frontend.

Zustand for global state management of language and category.

Responsive CSS for mobile-first design, ensuring usability on all devices.

Challenges I solved:

Multi-language support (English, Hindi, Telugu).

Dynamic filtering of jobs by category.

Responsive layout for desktop, tablet, and mobile.
