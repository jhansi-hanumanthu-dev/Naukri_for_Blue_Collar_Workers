# JobMitra - React Job Portal

## Overview
JobMitra is a responsive job portal built with **React** and **TypeScript**.  
It allows users to:  
- Select a language (English, Hindi, Telugu)  
- Browse job categories  
- View jobs in each category  
- See detailed job information and apply  

## Tech Stack
- React  
- TypeScript  
- Zustand (state management)  
- React Router DOM  
- CSS (responsive design)  


## Installation
1. Clone the repo:
```bash
git clone <repo-url>
cd JobMitra
2.Install dependencies:
npm install
3.Run the app:
npm start
Open http://localhost:3000
 in your browser
