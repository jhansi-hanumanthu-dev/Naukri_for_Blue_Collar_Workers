# JobMitra - React Job Portal

## Overview
JobMitra is a responsive job portal built with **React** and **TypeScript**.  
It allows users to:  
- Select a language (English, Hindi, Telugu)  
- Browse job categories  
- View jobs in each category  
- See detailed job information and apply  

## Tech Stack
- React  
- TypeScript  
- Zustand (state management)  
- React Router DOM  
- CSS (responsive design)
